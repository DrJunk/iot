What is IRPi?

IRPi is a universal IR remote based on Raspberry Pi. It allows you to control any IR-based device such as TVs, set-top boxes and air conditioners. IRPi controls mulitple devices from the cloud using a simple and intuitive interface.

Features

  • Record and Transmit any IR signal
  • Control your device through a mobile-friendly website
  • Manage multiple Raspberry Pi's from the cloud
  • Set timers for future remote commands
  • Access manually through easy HTTP Requests

Putting the IR in IRPi

IR is a light wave with a longer length than red light - this makes it invisible to the human eye. Since most day-to-day objects emmit IR signal naturally, IR remotes/receivers don't simply emmit and record IR; they use a unique protocol. Only when the transmitter modulates IR at a 38Khz frequency, the receiver detects a 'high' signal. This can be explained simply in the following diagram.

In order to read and transmit IR codes from the receiver, we used the GPIO library provided by microsoft.
For reading signals, we used the GPIOChangeReader class provided by microsoft. It uses DMA to record fast changes in the IR receiver signal, which we can later read from the device using the class.
Transmitting was a bigger challenge for us. Using software only to obtain an accurate and reliable 38Khz rate wasn't succesful. For that purpuse we used a '555 timer' connected to the IR LED on our breadboard.
